1. Antenna Diversity Boards

RX Boards at 2.4GHz

AD1_2.4GHz

Features: The receiver employs two antennas to select the best signal, a technique known as antenna diversity. This ensures reliable communication by mitigating signal loss caused by interference or obstacles.

RX Boards at 900MHz

2. True Diversity or Full Diversity Boards

RX Boards at 2.4GHz

TD1_2.4GHz

Features: In this design, the receiver utilizes two independent Semtech LoRa chips, each connected to its own antenna and operating on the same frequency. This configuration provides full diversity for the receiver, improving signal reception and overall reliability.

RX Boards at 900MHz

TX Boards (Shared for Single Antenna, Antenna Diversity & True Diversity)

Note:
  • The TX boards in this section are shared across Single Antenna Mode, Antenna Diversity, and True Diversity configurations.
  • They are not duplicated under each category to avoid redundancy.
  • They are optimized for 2.4GHz and 900MHz frequencies.
  • Gemini Mode (GM) and Gemini Xrossband (GemX) have their own TX boards, which are listed separately.

TX Boards at 2.4GHz

TX1_2.4GHz

Features: A basic transmitter module without an OLED display or TX Backpack for simple and reliable 2.4GHz communication.

TX2_2.4GHz

Features: This transmitter integrates an SSD1306 OLED display for telemetry and configuration adjustments but does not include a TX Backpack.

TX3_2.4GHz

Features: The most advanced module with both an SSD1306 OLED display and a TX Backpack, allowing seamless communication with GCS and external devices.

TX Boards at 900MHz

3. Gemini Mode Boards

RX Boards

Note:
  • The Gemini RX boards are actually the True Diversity RX boards operating in Gemini Mode.
  • To enable Gemini Mode, set the RX Antenna Mode to Gemini via Lua script.
  • Single Antenna RX boards can also operate in Gemini Mode but must use Model 5 in Model Config Matching.
  • Gemini-capable receivers should use Model 6 in Model Config Matching.
  • Click on 🔗 Go to True/Full Diversity RX Boards to view compatible receivers.

Frequently Asked Questions

Q: What protocols does the system support?

A: The system supports ELRS, mLRS, and MAVLink protocols for reliable telemetry and control.

Q: Can I use these boards for long-range communication?

A: Yes, the board designs are optimized for long-range communication with minimal latency.

Q: Are there future updates planned?

A: Absolutely! We’re planning a version 2.0 with improved PCB trace impedance, refined ground loops, and enhanced EMI performance.

About Us

Receiver and transmitter solutions for Ground Control Stations (GCS), designed to provide reliable telemetry and control using ELRS, mLRS, and MAVLink protocols. This repository features a wide range of innovative board designs optimized for long-range communication, minimal latency, and robust performance under varying conditions. From Antenna Diversity to True Diversity and Gemini Mode boards, these designs cater to diverse operational needs and ensure seamless integration with UAV systems.
